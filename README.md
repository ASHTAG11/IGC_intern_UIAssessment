# Weather App (UI Intern Assessment)

# Date : 25.02.2026

# Location : KCE @NEW HALL 2

## Project Overview
This is a simple web-based Weather Application developed as part of the UI Intern Assessment.  
The application allows users to enter a city name and view real-time weather information such as temperature, humidity, pressure, and weather conditions using the OpenWeather API (FREE) .

The project is built using HTML, CSS, and JavaScript and focuses on consuming REST APIs and displaying data in a user-friendly interface.

---

## Setup / Run Instructions

Follow these steps to run the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/weather-app-ui.git

2. Open the project folder in VS Code.

3. Open main.html using Live Server

4. Make sure you have an active internet connection (required for API calls).

5. Enter a city name and click Get Weather to view the data.


## API Details

This project uses the OpenWeather Current Weather API.
